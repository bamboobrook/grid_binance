# Membership And Payment

Understand plan pricing, grace period behavior, and exact-amount payment rules.

Membership is required before any new strategy can start.

Each payment order must be completed with the exact chain, token, and amount shown in the billing center. Overpayment, underpayment, or wrong token requires manual review.

Renewal stacking is allowed, and an expired membership enters a 48-hour grace period before running strategies are auto-paused.
