# 币安 API 设置

在运行任何策略前，请先安全创建并绑定币安 API 凭证。

登录后进入 `/app/exchange`。这里是保存币安凭证、查看掩码结果、执行连接测试的唯一入口。

创建币安 API 时，只开启“读取”和“交易”权限，绝对不要开启提现权限。

在 `/app/exchange` 中保存 API Key 和 Secret。Secret 只会保存在服务端，并且前端只显示掩码后的 Key。

保存后，务必运行一次连接测试，确认现货、U 本位、币本位权限以及合约所需的对冲模式都符合要求。

## 实盘测试说明

- 平台不会从服务器的 `.env` 里读取你的币安 API。每个用户都必须在 `/app/exchange` 内单独保存自己的密钥。
- 如果你的币安 API 做了 IP 白名单，请填写这台机器对外访问时使用的公网 IP。
- 第一次实盘建议先从现货和极小资金开始，再逐步测试合约权限。
- 每次保存新密钥后，都要重新运行连接测试，再去创建或重启策略。
