# 安全中心

## 功能范围

通过 `/app/security` 管理密码和 TOTP，不会暴露任何交易所 Secret。

## 包含内容

- 修改密码
- 启用 / 停用 TOTP
- 查看账户安全状态
- 查看币安凭证是否仍为掩码

## 安全规则

- 注册完成后可以直接登录
- 建议为本平台使用独立强密码
- 用户和管理员都支持 TOTP
- 管理员账号必须启用 TOTP
- 密码修改和 TOTP 操作都必须通过已登录的 POST 请求完成
- 币安 API Secret 始终加密存储，并保持掩码显示
- 币安 API 必须关闭提现权限

## 推荐操作顺序

1. 登录 `/login`
2. 打开 `/app/security`
3. 绑定币安 API 前，先启用 TOTP
4. 妥善保存验证器里的恢复资料
5. 保存币安凭证后，再回来看一遍是否仍是掩码状态

## 什么时候要回来检查

- 刚做完密码重置
- 更换设备或验证器之后
- 准备切换币安凭证之前
- 准备开始合约实盘之前
