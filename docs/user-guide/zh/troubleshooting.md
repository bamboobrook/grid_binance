# 故障排查

## 登录与安全

- 如果登录提示 `admin totp setup required`，先打开 `/admin-bootstrap` 创建管理员 TOTP，再回到登录页输入当前验证码
- 如果普通用户登录提示 `totp code required`，请输入验证器当前的 6 位 TOTP 验证码

## 交易所问题

- 如果平台提示 `API credentials invalid`，请打开 `/app/exchange` 查看具体校验失败项
- 常见原因包括：时间偏差、权限不足、误开提现权限、市场权限不匹配、未开启对冲模式

## 策略问题

- 如果启动失败，请到 `/app/strategies/:id` 查看预检失败步骤
- 如果停止状态一直卡在 `停止中`，说明平台还在等待交易所完成撤单或市价平仓回执
- 如果删除被阻止，请先清空工作挂单和剩余仓位

## 计费问题

- 支付金额必须完全一致
- 转错币种、转错金额或异常转账都会进入人工复核
- 如果订单长时间停在 `confirming`，请等待对应链路达到确认数
