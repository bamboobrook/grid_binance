# 创建网格策略

建议先在 `/app/strategies/new` 创建草稿，再决定是否启动。

## 草稿创建流程

1. 在交易对搜索框里模糊搜索币种，并点击结果锁定交易对。
2. 选择市场类型和策略模式。
3. 选择生成方式：
   - 等差：按固定百分比间距生成。
   - 等比：按复合比例生成。
   - 完全自定义：每个网格都手动编辑。
4. 选择编辑方式：
   - 批量生成：适合快速搭建阶梯。
   - 逐格自定义：适合直接在界面里逐格修改。

## 参考价

- `参考价来源` 支持两种方式：手动输入、当前市价。
- 手动输入会使用你填写的参考价格。
- 当前市价会在保存草稿前拉取币安最新价格作为参考价。
- 如果你已经切到逐格自定义，参考价更多用于预览和新增层级时的辅助，不会强行覆盖你手动改好的每一格。

## 金额模式与逐格编辑

- `按 USDT` 适合让每格保持接近相同资金占用。
- `按币数量` 适合让每格保持相同币数。
- 现货经典网格和现货只卖模式，在启动前还需要有足够的基础币库存来支撑卖出网格。
- 在 `逐格自定义` 模式下，你可以直接逐行设置：
  - 入场价
  - 与上一格间距
  - 单格投入金额 / 单格币数量
  - 网格止盈范围
  - 追踪止盈范围
- 页面内部仍会把这些内容序列化为 `levels_json`，但正常使用时不需要你手动写 JSON。

## 批量阶梯参数

批量模式主要使用这些参数：

- `参考价格`
- `网格数量`
- `批量间距 (%)`
- `网格止盈 (%)`
- `追踪止盈 (%)`

如果你只是想快速生成一套阶梯，用批量模式最快；如果你要逐格控制每一格，就切到逐格自定义。

## 收益与风控

- `整体止盈 (%)`：当当前有效持仓整体达到设定收益目标时，整套策略触发整体止盈。
- `整体止损 (%)`：可选，留空表示不启用整体止损。
- `追踪止盈 (%)`：可选；启用后会改成 taker 方式止盈，手续费通常高于 maker。
- 追踪止盈不能大于对应网格的止盈范围。

## 模板

- `应用模板` 会把管理员预设复制成你自己的策略草稿。
- 复制完成后，草稿归你自己所有。
- 后续模板更新不会覆盖你已经应用过的策略。

## 启动前要求

- 创建草稿并不会直接启动策略。
- 预检会检查会员状态、交易所 API、交易对元数据、余额 / 保证金以及合约对冲模式等条件。
- 运行中的策略不能热修改参数。
- 必须先暂停，再保存修改，再重新预检，最后再启动或恢复。
