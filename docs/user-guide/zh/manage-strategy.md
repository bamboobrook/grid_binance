# 管理策略

## 主要页面

- `/app/strategies`：查看草稿、运行中、暂停中和异常策略，并执行批量动作
- `/app/strategies/:id`：进入单个策略工作台，查看预检、运行事件和编辑入口
- `/app/orders`：查看成交、交易所成交记录和导出
- `/app/analytics`：查看账户级与策略级收益、手续费、资金费和钱包快照

## 生命周期规则

- 新建后默认是 `草稿`
- 启动前一定会先执行预检；任何步骤失败都会阻止启动
- 暂停会撤销工作挂单，但保留持仓或合约仓位
- 恢复时会重新预检，并基于当前持仓上下文重建策略
- 停止时如果没有剩余持仓，会直接进入 `已停止`
- 如果还有剩余持仓，会先进入 `停止中`，直到交易所侧真正平仓完成
- 删除前必须确认挂单和剩余仓位都已清空

## 编辑规则

- 运行中的策略不能热修改
- 必须先暂停，再保存参数，再重新预检，最后重启
- 批量动作支持：`启动所选`、`暂停所选`、`删除所选`、`全部停止`

## 故障提示

- 预检失败时，页面会明确告诉你是哪一步失败
- 如果交易所凭证失效，平台会同时发站内通知和 Telegram 提示
- 如果运行时对账失败，策略会自动暂停，并记录修复建议
